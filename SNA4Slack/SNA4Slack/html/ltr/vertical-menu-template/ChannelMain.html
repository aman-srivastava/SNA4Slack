Channel Statistics

Channel Creation Table

About

This table shows stats specific to the team, considering all channels.

Description

1. Total Messages: Sum of all the messages in each channel.

2. Average Messages: Average number of messages in each channel.

3. Max Messages: Maximum number of messages in a channel.

4. Least Messages: Least number of messages in a channel.

5. Total Channels: Total number of channels in the team.

6. Total URLs: Total number of URLs in the team.

7. Total Emoticons: Total number of emoticons in the team.
